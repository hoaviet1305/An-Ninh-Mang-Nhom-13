RSA Demo WinForms

Cách chạy:
1. Mở file RSA_Demo.csproj bằng Visual Studio.
2. Đợi restore.
3. Run.

Chức năng:
- Tạo khóa ngẫu nhiên
- Tạo khóa thủ công
- Ký văn bản
- Lưu chữ ký
- Chuyển tiếp dữ liệu
- Kiểm tra chữ ký
- Sửa đổi chữ ký
- Báo lỗi khi thiếu dữ liệu, p/q không nguyên tố, chưa chọn hàm băm, chưa có chữ ký, văn bản thay đổi, chữ ký thay đổi
